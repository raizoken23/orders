# **App Name**: ScopeSheet Pro

## Core Features:

- Digital Scope Sheet: Allows inspectors to fill out digital scope sheets with claim information.
- Roof Diagram Tool: Provides a tool for creating roof diagrams directly within the app.
- Claim Information Management: Manages and stores claim information for each inspection.
- Subscription Management: Handles user subscriptions, billing, and access control.
- Image Analysis Tool: AI powered tool analyzes uploaded roof images to detect potential issues. This tool uses LLM reasoning.

## Style Guidelines:

- Primary color: Dark Blue (#34495E) for professionalism and reliability.
- Background color: Light Gray (#ECF0F1) to ensure clarity and reduce eye strain during long use.
- Accent color: Orange (#E67E22) for CTAs and important interactive elements, providing contrast and drawing attention.
- Font pairing: 'Space Grotesk' (sans-serif) for headings to give a modern tech feel, paired with 'Inter' (sans-serif) for body text, optimizing readability.
- Use clear, professional icons representing different inspection aspects (roof, structure, etc.).
- A clean, intuitive layout with a tabbed interface for different sections (scope sheet, roof diagram, claim info).
- Subtle transitions and animations when navigating between sections and when data is loading to enhance user experience without distraction.
- The roof diagram tool will utilize precise lines and mathematical calculations to represent roof dimensions and angles. Display beneficial statistics like surface area and slope in an informative manner.
